# E-commerce Data Analysis

## Overview
This project analyzes e-commerce transactional data to understand sales performance, customer behavior, and revenue drivers.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## Data Cleaning
- Handled missing and inconsistent values
- Corrected data types
- Removed duplicates
- Treated outliers in total sales

## Key Analysis & Insights
- Identified high-value customers based on order frequency
- Analyzed monthly and yearly sales trends
- Ranked payment methods by total revenue
- Studied the impact of discounts on total sales
- Identified top-performing products and categories

## Power BI Dashboard
The dashboard visualizes:
- Total sales and revenue trends
- Customer segmentation
- Payment method performance
- Top products
